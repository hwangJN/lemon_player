# 🍋 LEMON PLAYER
🔗 [웹페이지 바로가기](https://lemon-player.fly.dev/)


## 프로젝트 소개 & 기능
- 뮤직 플레이리스트 & 플레이어 구현 웹사이트입니다
- 로그인된 유저는 자신만의 플레이리스트를 가질 수 있습니다
- 재생 수에 따른 차트를 확인할 수 있습니다
- 유저 프로필 수정이 가능합니다
- 등록된 뮤직비디오를 감상할 수 있습니다
- 반응형 웹페이지로 제작했습니다

## 제작 동기
- 노마드코더 클론 코딩 챌린지 참여
- 새로운 기술 스택 도전
